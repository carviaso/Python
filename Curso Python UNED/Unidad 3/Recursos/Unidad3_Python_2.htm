Introduccin

Todo software en la actualidad, tiene la necesidad de persistir los datos que trasciegan por el, para luego generar con estos datos informacin de valor.

Archivos, bases de datos relacionales o no estructuradas, son las maneras ms comunes de almacenamiento de datos. En esta unidad se desarrollar la manera en que Python interacta con archivos para guardar, modificar o leer datos.

Objetivos de aprendizaje

Objetivo General

  • Manipular diferentes archivos mediante una aplicacin para la exportacin de datos.

Objetivos especficos

  • Abrir un archivo aplicando diferentes opciones de apertura.
  • Crear un archivo que sobre escribe otro archivo existente.
  • Gestionar un archivo para leerlos, gestionar la data y guardar los datos nuevamente.
