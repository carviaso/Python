Introducción a la unidad 3
Introducción

Todo software en la actualidad, tiene la necesidad de persistir los datos que trasciegan por el, para luego generar con estos datos información de valor.

Archivos, bases de datos relacionales o no estructuradas, son las maneras más comunes de almacenamiento de datos. En esta unidad se desarrollará la manera en que Python interactúa con archivos para guardar, modificar o leer datos.

Objetivos de aprendizaje

Objetivo General

  • Manipular diferentes archivos mediante una aplicación para la exportación de datos. ​ ​

Objetivos específicos

  • Abrir un archivo aplicando diferentes opciones de apertura.
  • Crear un archivo que sobre escribe otro archivo existente.
  • ​ Gestionar un archivo para leerlos, gestionar la data y guardar los datos nuevamente.
Contenidos
  • Creación de archivos
  • Gestión de archivos
